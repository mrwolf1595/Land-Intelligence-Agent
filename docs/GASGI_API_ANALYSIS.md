# تحليل المركز الوطني للمعلومات الجيومكانية (GASGI API)
**تاريخ التحليل:** أبريل 2026

تنفيذاً لطلبك، توجه وكيل المتصفح لمنصة **"الخارطة التفاعلية"** التابعة للمركز الوطني للمعلومات الجيومكانية (الجيوبورتال الوطني)، وتم تحليل حركة مرور البيانات والطبقات (Layers) المتاحة.

---

## 1. بنية المنصة العامة (Architecture)
تعمل المنصة عبر تطبيق خرائط مبني بـ `ArcGIS Experience Builder` ومرفوع على النطاق `geoportal.sa`. 
مثلها كمثل منصة (بلدي)، تعتمد بشكل مطلق على واجهات REST API الخاصة بـ Esri لتغذية الخريطة بالطبقات بدلاً من النصوص العادية.

---

## 2. المسارات المكتشفة (Key Feature Servers)
المنصة منظمة بشكل ممتاز وتوفر خوادم معالم (Feature Servers) تحت مظلة الـ NDM (نموذج البيانات الوطني):
* **استخدامات الأراضي (Land Use):** `https://www.geoportal.sa/pub/rest/services/NDM/LandUse/FeatureServer`
* **قطع الأراضي (Land Parcels):** `https://www.geoportal.sa/pub/rest/services/NDM/LandParcel/FeatureServer`
* **المناطق الإدارية (Admin Region):** `https://www.geoportal.sa/pub/rest/services/NDM/AdminRegion/FeatureServer`

---

## 3. آلية جلب البيانات (Payload & Request)
لدمج هذا مع كود الذكاء الاصطناعي (Agent) الذي يبحث عن تقييم أرض، يمكن إرسال طلب `POST` لهذه الروابط مع تحديد المعاملات الجغرافية (Spatial Rel):
* `geometryType`: استخدام `esriGeometryPoint` لتمرير طول وعرض اللوكيشن.
* `spatialRel`: تمرير `esriSpatialRelIntersects` لطلب البيانات التي تتقاطع مع نقطة البحث.
* `outFields`: تمرير `*` لإرجاع كافة سمات الأرض كصيغة JSON.

## 💡 التوصية لتعديل `gasgi/scraper.py`:
هذه الواجهة كنز حقيقي! يمكنك استغلالها لإنشاء نظام التحقق الجغرافي.
ستكتب ਦالة `requests.post` تُرسل الإحداثية الواردة، وبناءً عليه ستُرجع الخوادم نوع الأرض والتنظيم الجغرافي المحيط بها بدقة مكانية وطنية موثوقة. يمكن استخدام هذه البيانات لتغذية المحللات المالية بمعلومات مثل البعد عن شبكة السيول أو موقع الأرض من شبكة المطورين.
