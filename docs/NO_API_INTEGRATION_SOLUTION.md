# 💡 الحل الفني: جلب البيانات الحكومية بدون وصول رسمي (APIs)

في حال عدم وجود وصول رسمي (API) للوزارات والهيئات بسبب الشروط التقنية ومحوددية الوصول للشركات الكبرى، فإن **"مدير الذكاء المالي - CFO"** لديه حلول التفافية وذكية لجلب هذه البيانات بشكل آلي ومستمر، وذلك لضمان جودة اتخاذ القرار دون أن يتوقف النظام.

---

## 🏛️ 1. بيانات وزارة العدل (المؤشرات العقارية)
**المشكلة:** لا يوجد API لوزارة العدل للأفراد.
**الحل (Web Scraping):** 
وزارة العدل توفر بوابة عامة للمجتمع تسمى "المؤشر العقاري" (moj.gov.sa/ar/ministry/departments/notary/pages/indicators.aspx).
* **التنفيذ:** سنقوم بكتابة سكريبت بسيط (`sources/moj/scraper.py`) باستخدام `curl_cffi` (نفس الأداة التي نستخدمها الآن لاختراق منصة Wasalt).  يقوم السكريبت مرة كل أسبوع بزيارة هذه الصفحة العامة، اختيار المدينة والحي من القوائم المنسدلة، ثم يستخرج "متوسط سعر المتر المفرغ فعلياً" ويسجله في الداتابيس.

## 📊 2. بيانات الهيئة العامة للإحصاء (مواد البناء)
**المشكلة:** الهيئة تصدر البيانات في ملفات إكسل (Excel) وتقارير PDF شهرية.
**الحل (Data Parsing):** 
* **التنفيذ:** إنشاء برنامج (Bot) يراقب الرابط العام لنشرات الهيئة. بمجرد نزول "النشرة الشهرية لمتوسط أسعار السلع"، يقوم البرنامج بتنزيل ملف ה־Excel للبيانات الخام (والذي توفره الهيئة للعموم مجاناً)، ثم يقرأ منه برمجياً باستخدام مكتبة `pandas` ليستخرج سعر "الحديد" و"الأسمنت" ويحدث متغيرات `CONSTRUCTION_COST` في قاعدة البيانات فوراً.

## 🏦 3. بيانات البنك المركزي (مؤشر السايبور SIBOR)
**المشكلة:** بيانات الفائدة تتغير وتحتاج اتصال مع البنوك لتأكيدها.
**الحل (Financial API/Scraping):** 
هناك طريقتان لجلب الأسعار الفائدة الحالية:
* **الطريقة الأولى:** عمل Scraping بسيط لصفحة أسعار الفائدة في موقع البنك المركزي (SAMA)، والمتاحة للعموم كجدول HTML، وسحب نسبة الـ 3 شهور والـ 6 شهور والـ 12 شهراً.
* **الطريقة الثانية:** الاعتماد على APIs عالمية مجانية للبيانات الاقتصادية (مثل TradingEconomics أو Investing.com) وتسجيل سعر (Saudi Arabia Interbank Rate) منها دورياً.

## 🗺️ 4. معلومات الأحياء والخدمات (البديل للبلديات)
**المشكلة:** خدمات أمانات المدن تتطلب وصول GIS مغلق، وتحتاج API خاص لتحديد اشتراطات البناء وخدمات الشوارع.
**الحل (OpenStreetMap - OSM):** 
* **التنفيذ:** استخدام واجهة (Overpass API) المجانية والمفتوحة المصدر بالكامل (لا تتطلب مفاتيح سرية، ولا حسابات ولا يوجد قيود). سنرسل للإحداثيات الجغرافية التابعة للإعلان وسيقوم Overpass بإرجاع قائمة بكافة المدارس، الجوامع، محطات المترو، الأسواق والمستشفيات المحيطة بالعقار بقطر 3 كيلومتر لمعرفة "مدى حيوية الموقع".
